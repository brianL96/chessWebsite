Rules

Welcome to my website. Here is what you need to know to get started:

  • To access most features, you need to sign up for an account and then log in.

  • The email you enter is not verified: entering a unique and valid string of characters is sufficient.

  • To make demonstrating PvP easier, this website uses Session Storage instead of Local Storage. If you'd like to test PvP features, you're likely to have to create a second account. You can then log into each account from a seperate tab in your browser (probably easiest and quickest method), or you can use a different browser for each account, or you can log on from different computers. Creating an account is very quick and easy.

Features:

  • Users can play four versions of chess.

  • Users can concurrently play in up to four rooms, on condition that each of those rooms is in a different game mode.

  • This application will keep track of personal stats for each game mode, including an ELO score.

  • Players are ranked according to ELO score on the Leaderboards Page.

  • Users can save games, replay or edit them, and save the edits.

A user with the right credentials can log into the Admin Page. From there they can:

  • View general information - the date and time the server started up, the last time an admin logged in, how many times an admin has logged in since the server started, etc.

  • View particular game mode information - the most amount of concurrently fully occupied rooms, the amount of time spent at that highest occupation, etc.

  • Index and display specific room information, including currently occupying players, current game status, the amount of rematches played, the date and time the first match ended, etc.

  • Turn room(s) off - which prevents a user from entering that room, and players currently inside that room cannot set up a rematch.

  • Turn room(s) on.

  • Kick players from room(s).

  • Adjust the amount of rooms a particular game mode has open to users.
